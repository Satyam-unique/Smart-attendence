📡 RFID-Based Smart Attendance System
An IoT-based Smart Attendance System that uses RFID technology + ESP32 + Web Dashboard to automate attendance marking in classrooms.

🚀 Project Overview
This project replaces traditional manual attendance systems with an automated, fast, and accurate solution using RFID cards.

📶 RFID cards are scanned using RC522 module
⚡ ESP32 processes and sends data via Wi-Fi
🌐 Web dashboard displays attendance in real-time
📊 Faculty and students can monitor attendance easily

🎯 Objectives
Automate attendance marking
Reduce manual errors and time consumption
Provide real-time attendance tracking
Prevent proxy attendance
Build a scalable IoT-based solution
🛠️ Tech Stack
🔌 Hardware
ESP32 Microcontroller
RC522 RFID Module
RFID Cards/Tags
💻 Software
Arduino IDE
HTML, CSS, JavaScript
Firebase (Realtime Database / Firestore)
Web APIs
⚙️ System Architecture
RFID Card → RC522 Reader → ESP32 → WiFi → Firebase → Web Dashboard
🔄 Working Principle
Student scans RFID card
RC522 reads unique UID
ESP32 processes UID
Data sent to server via Wi-Fi
Server verifies student
Attendance is marked
Dashboard updates in real-time
